## 📌 Project Overview

This repository contains my completed **Data Analytics Internship Project** submitted as part of the **Cognify Technologies Internship Program**.

The project focuses on cleaning, exploring, analyzing, and visualizing restaurant data to uncover meaningful business insights using Python and data analytics libraries.

---

## 🎯 Objectives

* Perform data cleaning and preprocessing
* Conduct exploratory data analysis (EDA)
* Analyze restaurant ratings and customer preferences
* Identify trends and patterns in the dataset
* Create informative visualizations
* Generate actionable business insights

## ▶️ How to Run

### 1. Clone the repository

```bash
git clone https://github.com/your-username/cognify-technologies-data-analytics.git
```

### 2. Navigate to the project folder

```bash
cd cognify-technologies-data-analytics
```

### 3. Install required libraries

```bash
pip install -r requirements.txt
```

### 4. Run the Python script

```bash
python source-code/data_analysis.py
```

Or open the Jupyter Notebook:

```bash
jupyter notebook notebooks/Data_Analysis.ipynb
```
